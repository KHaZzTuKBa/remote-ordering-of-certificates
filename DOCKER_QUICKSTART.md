# Быстрый старт с Docker

## Запуск всего приложения одной командой

```bash
docker-compose up -d
```

## Доступ к приложению

- **Фронтенд:** http://localhost:4200
- **Бэкенд API:** http://localhost:5066
- **Swagger:** http://localhost:5066/swagger

## Остановка

```bash
docker-compose down
```

## Просмотр логов

```bash
docker-compose logs -f
```
