# Фронтенд приложения "Удаленный заказ справок"

Angular приложение для удаленного заказа справок об обучении.

## Требования

- Node.js (версия 18 или выше)
- npm или yarn

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

1. Запустите приложение в режиме разработки:
```bash
npm start
```

2. Откройте браузер и перейдите по адресу: `http://localhost:4200`

## Структура проекта

- `src/app/pages/` - страницы приложения
- `src/app/services/` - сервисы для работы с API
- `src/app/models/` - модели данных (DTO, Response)
- `src/app/app.routes.ts` - настройка роутинга

## API

Приложение настроено на работу с бэкендом по адресу `http://localhost:5066`. Убедитесь, что бэкенд запущен и доступен.

## Особенности

- Используется Angular 17 с standalone компонентами
- Роутинг настроен для всех необходимых страниц
- Обработка ошибок реализована во всех сервисах
- Применена указанная цветовая палитра
- Используются шрифты Roboto и Inter
